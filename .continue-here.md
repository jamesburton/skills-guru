<decisions_made>

- Chose Node.js (built-ins only, no npm deps) for scripts — universal in Claude Code environments
- Split shareable memory from private .local/ config — enables sharing without leaking secrets
- Secret identifiers use stable format {{SECRET:fileHash:keyPath:seq}} for round-trip editing
- 8h inactivity + 24h absolute TTL with heartbeat for session secrets
- SKILL.md stays under 500 lines (76 actual) with 7 reference files loaded on demand
- "Merge" option removed from conflict handling — skill files don't merge well, manual diff instead
- Script sandboxing uses BLOCK/WARN/INFO severity levels with defined pattern-to-severity mapping
</decisions_made>
